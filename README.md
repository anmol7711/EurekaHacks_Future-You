🧩 How It Works

The system converts habits into normalized scores and combines them into life outcomes.

// Example (simplified)
health = sleep * 0.3 + exercise * 0.25 + diet * 0.2
       - screen * 0.1 - stress * 0.15;

These scores drive:

Timeline generation
Visual metrics
Final trajectory score
